Simple Interactive Marker Example

Run the following commands in the terminal then refresh this page.

  1. roslaunch nimbus_bringup nimbus_bringup.launch
  2. roslaunch nimbus_web_interfaces web_services.launch
  3. rosrun interactive_marker_proxy proxy topic_ns:=/nimbus_interactive_manipulation target_frame:=/jaco_base_link
  4. rosrun tf2_web_republisher tf2_web_republisher
  5. roslaunch rosbridge_server rosbridge_websocket.launch
